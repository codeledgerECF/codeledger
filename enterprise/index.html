Five governance planes

Each plane answers a different stakeholder's question.

Enterprise rollout starts where your most urgent problem lives — spend control, risk management, compliance evidence, or onboarding efficiency. Each plane can be unlocked incrementally. Depth is access-controlled per role.

01

Context

Engineering leadership

“What did the agent actually use — and why?”

Shows exactly which files entered the agent prompt, how each was scored, and what the Context-Compiler trimmed, hoisted, retained, or skipped. Every decision is traceable.

Evidence artifacts

Active bundle · Optimization trace · Broker timeline · Context delta

02

Verification

Security / Release management

“Is this safe to ship?”

Turns every PR, CI run, and proof pack into structured release evidence. Risk signals, drift flags, and evidence gaps surface before code merges — not after it deploys.

Evidence artifacts

Validation summary · PR risk check · Proof pack · Catch-rate history

03

Governance

Compliance / Legal / Internal audit

“What can we hand the auditor?”

Maps policies, access rights, and control coverage into exportable evidence packages. Signed Truth Audit certificates record what the agent touched without exposing source code.

Evidence artifacts

Access model · Control map · Signed certificates · Audit export

04

Fleet

Platform / CTO office

“Who is doing what, across every agent and every team?”

Coordinates multi-agent work across teams with file claims, leases, and per-agent scorecards. Leadership gets a live view of AI activity, output quality, and hours recovered.

Evidence artifacts

Agent scorecards · File claims · Leases · Hours-saved proof

05

Learning

HR / Onboarding / Platform

“How do new engineers start with institutional knowledge instead of a blank slate?”

Validated patterns, accepted changes, and architectural decisions are persisted across sessions. Every new developer — and every agent — starts with the context your senior engineers built.

Evidence artifacts

Pattern promotions · Lessons · Replayable outcomes · Onboarding baseline
