What it does

Everything a team needs to ship AI-assisted code safely

Deterministic context, not guesses

Every file is scored using 6 weighted dimensions: keyword match, dependency centrality, churn recency, test presence, and more. The same task produces the same bundle every time.

Collective Intelligence — Prompt Coach

Every task prompt is evaluated against a curated signal library before your AI agent runs. Missing chain-of-thought? Scope contradiction? The Prompt Coach catches it first — and learns from your team's outcomes over time.

Architecture guardrails in CI

Risk, drift, and evidence-gap signals on every PR. Smart gates conditionally run heavy checks only when risk is High — keeping CI fast by default.

Shared team memory

The Team Context Ledger persists accepted changes, validated surfaces, and terminology definitions across sessions. Every agent starts with shared context, not a blank slate.

Engineering Intelligence Dashboard

Per-agent scorecards, destabilization metrics, hours saved, and proof-of-capability evidence — all derived from actual events, not estimates.

Local-first, offline capable

Context selection and license verification run entirely on your machine. No source code leaves your environment. Works fully offline after activation.
